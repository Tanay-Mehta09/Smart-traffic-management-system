# 📖 Overview

The **Smart Traffic Management System** is an Artificial Intelligence-based intelligent traffic control solution designed to reduce urban traffic congestion through real-time vehicle detection and adaptive traffic signal management.

Unlike traditional traffic lights that operate using fixed timers, this system continuously analyzes traffic density using **Computer Vision** and **Deep Learning** models to dynamically adjust signal timings, improving traffic flow while reducing congestion, fuel consumption, travel time, and carbon emissions.

This project contributes towards building safer, smarter, and more sustainable cities by leveraging AI-driven decision-making.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Tanay-Mehta09/Smart-traffic-management-system.git
```

Move into the project

```bash
cd Smart-traffic-management-system
```

Install dependencies

```bash

```

Run the notebook

```bash
jupyter notebook
```

or open it directly using **Google Colab**.
